# Passport OCR (Optical Character Recognition)
Extract user data from a passport image.

Extracted info includes:
- Surname
- Name
- Sex
- Date of Birth
- Nationality
- Passport type
- Passport number
- issuing country
- Expiration date
- Personal number

**Note:**
  Personal numbers aren't used in all country.
